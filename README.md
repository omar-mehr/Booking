# 🏡 Booking System

This is a booking system that manages reservations for cabins, rooms, and tents.


---

## ✨ Features

- 🏠 Book cabins
- 🛏️ Book rooms
- ⛺ Book tents
- 📋 Manage reservations

---

## 🌐 Language

The application interface and system are in Swedish.

---

## 🧠 Architecture

This system is developed using object-oriented programming (OOP) in C#.

---

## 📦 Installation & Running the Project

Follow these steps to download and run the application locally.


### 1. Clone the repository

```bash
git clone https://github.com/omar-mehr/Booking.git
```

### 2. Navigate into the project folder

```bash
cd Booking
```

### 3. Build the project


```bash
dotnet build
```

### 4. Run project

```bash
dotnet run
```
